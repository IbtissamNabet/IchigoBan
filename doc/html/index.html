Introduction

ICHIGO'BAN est en jeu de puzzle inspiré du jeu classique SOKOBAN, dans lequel un gardien doit pousser des fraises jusqu'à leur emplacement cible.
Le jeu est composé de plusieurs niveaux (6 dont un niveau d'initiation) avec des configurations de plus en plus difficiles. Il a été développé par:

Compilation

Se compile avec un Makefile Il suffit de taper la commande suivante:

$ ./bin/sdlJeu

$ ./bin/txtJeu
